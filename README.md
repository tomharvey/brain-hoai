# Head of AI Operating System

A markdown-native personal operating system for leading AI strategy across an insurtech company.

## Quick start

1. Open in Claude Code: `claude` in the repo root.
2. Available slash commands: `/morning`, `/manifest`, `/decision`, `/initiative`, `/review`, `/ingest`.

## Design principles

- **Markdown is truth** — all knowledge lives in plain text files with YAML frontmatter.
- **Git is the database** — version history, branching, and collaboration via standard git workflows.
- **AI is collaborator, not dependency** — the vault works without AI; AI makes it faster.
- **Structure follows need** — add structure when it earns its keep, not before.
- **Link liberally** — wikilinks between notes surface connections over time.

## Tools

- **QMD** — local full-text search across the vault.
- **Claude Code** — agentic operations: create notes, triage inbox, generate reviews, rebuild the manifest.
