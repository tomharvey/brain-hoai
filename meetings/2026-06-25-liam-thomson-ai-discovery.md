## Key themes

### AI tool stack

- Switched from ChatGPT to Claude for almost everything over the last ~6 weeks
- Uses Claude Chat for strategic advisory and document creation; Claude Code for fast, local edits
- Created a "dashboard scope" prompt: uploads any doc (transcript, Notion page, etc.) and gets a structured visual summary for calls

### Document workflow — iterative branded docs

- Main use case: branded docs, pitches, and Notion pages
- Process: iterative back-and-forth in Claude Chat, feeding in feedback from conversations directly
- Admiral marketing budget pitch at version 27 (each Claude revision counts as a version)
- Notion pages now include granular detail: email subject lines, send schedules, sender attribution — Jade product launch page cited as a strong example

### Budget tracking with Claude + Chrome MCP

- Previously relied on Finance for budget updates; small spend was hard to track
- Built an Excel tracker: uploads bank/card transcripts monthly, Claude uses Chrome MCP to update the spreadsheet automatically
- Flags ~10 uncertain transactions for manual categorisation; Claude learns categories over time
- Result: always has a ~30-day accurate view of marketing spend without bothering Finance

### Copywriting — limitations and workarounds

- AI copywriting quality hasn't meaningfully improved since ChatGPT launch (~2–3 years)
- Key "AI-ism" to watch: overuse of the "power of three" sentence structure — fix: highlight instances, prompt Claude to vary structure (questions, statements, different lead words)
- Uses separate Claude Projects for email copy vs. LinkedIn posts to keep styles distinct
- Better results from tight, specific prompts than free-reign generation
- Has built up speed at manual edits; hasn't yet invested time in a formal tone-of-voice skill

### Claude Design for high-stakes presentations

- Moved away from the deck builder toward Claude Design for high-stakes decks
- Feeds in a communication playbook (built from an Ed session) plus source content (Notion pages, etc.)
- Output: stronger narrative structure where slide titles alone tell the full story — aligns with what Ed and Antton prioritise
- Slower and more token-intensive than deck builder, so reserved for key stakeholder presentations
- Plans to explore animation in Claude Design today
