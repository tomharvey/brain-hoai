**Date**: 2026-03-26 15:37
**Attendees**: [[abs]], Tom

## Key themes

### Current AI development challenges
- Pattern of replicating existing tools (Retool) rather than innovating
- Haulage project became Retool replica using latest services — underwriters report slower performance
- Default question is "Does Retool do this?" — should be "What's new/different we can build?"

### Successful vs unsuccessful project patterns
- **Referrals** (successful): greenfield, started with spikes and investigation, explored multiple ideas, didn't follow existing manual processes
- **Vehicle resolution** (unsuccessful): built on top of existing complexity, made process more complicated, no fundamental rethinking

### AI development philosophy
- 10x faster = 10x better outcomes, not 10x more features
- Opportunity to try multiple code structures/approaches — build 20 versions, pick best one
- Focus on what AI makes possible, not just faster
- Document business processes for agent automation
- Eliminate "janky spreadsheets" dependencies
