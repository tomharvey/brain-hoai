## Key themes

### Matt Lees — chief of staff model (enterprise prospecting)
- Shifted from reactive tools (one per prospecting stage) to a single Claude "chief of staff" project
- Fed: role profile, quarterly OKRs, weekly goals, buying signal criteria, stakeholder markdown files
- Now proactive sweeps of Granola transcripts, emails, Slack; cites sources (HubSpot, Granola, web URLs)
- "Anton test": does this activity move the needle on OKRs?
- BLUF (Bottom Line Up Front) on all outputs
- Results: moved from single-digit email conversion to meaningful pipeline movement (engaged meetings, watch list movements)
- Still needs nudging toward strategic thinking; not fully autonomous

### Aris — scheduling and recruitment automation
- Started with daily "morning Claude" chat → evolved to automated co-work prompt at 9am
- Daily agenda: interviews, priority actions, pipeline status, chases — 85–90% accuracy, improves with feedback
- Working primarily from Claude rather than switching between tools
- Marks tasks as done; Claude picks up completion from calendar/email sweeps

### Allen — leadership coaching project
- Fed 1:1 transcripts, manager feedback, external coaching notes into Claude project
- Creates scoring on listening, assertion, strategic clarity
- Provides specific sentence examples and reframing suggestions
- Identifies patterns across multiple meetings
- Brutal but useful; scores can be contextualised (CEO meeting vs. external presentation)

### Jake Wood — underwriting dashboard
- Built HubSpot-style deal prioritisation dashboard at AI breakfast; done by 10:30am
- Wants to extend to team (Billy, Curtis); challenge is getting others past the mindset shift, not just handing over the tool

### Strategic AI partnership vs. task automation
- Key shift: telling Claude the objective, not "click this, do this"
- Matt: moved from "tell it what to do" → "achieve this, figure out how"
- At what point does automation become a thinking partner? Still needs human nudging, but the framing matters

### Cross-team applications discussed
- BDM: broker relationship prioritisation at scale (Ollie's broker pulse)
- Underwriting: deal status summaries for broker call prep (Jake)
- Sales playbook coaching for consistent messaging
